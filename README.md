# StyleGen - Fashion & Outfit Advisor

A full-stack MERN application for creating, organizing, and discovering fashion outfits.

## Features

- User authentication (register/login)
- Create and manage outfits
- Browse community outfits
- User profiles with preferences

## Tech Stack

- MongoDB: Database
- Express.js: Backend framework
- React: Frontend library
- Node.js: Runtime environment
- Styled Components: Styling

## Setup Instructions

1. Clone the repository
2. Install dependencies: `npm run install-all`
3. Set up environment variables in `backend/.env`
4. Start MongoDB locally or provide connection string
5. Run the application: `npm run dev`
